#Lettus Plant Grow System
### Requirements:
- Make sure you are using node version 8
##### Install the packages:
$ npm install

Start the webpack server with live reload:
- `$ npm run start` for windows, mac, linux or Gitpod.
- `$ npm run c9` for Cloud 9 Users.
